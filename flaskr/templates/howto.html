Path Beschreibung
/categorize
Kategorisiert eine Transaktion, welche via HTTP-POST im JSON Format gesendet wird.

Folgendes Format muss genutzt werden, wobei text, usage, owner und amount Pflichtfelder sind:

{
"booking_date":" ",
"valuta_date":" ",
"text":" ",
"usage":" ",
"creditor_id":" ",
"owner":" ",
"iban":" ",
"bic":" ",
"amount':" "
}

Die Antwort wird in folgendem Format geliefert:
{
"category": "",
"probability":
}
